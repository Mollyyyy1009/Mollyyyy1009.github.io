数据库相关概念

数据库

​ 数据库是存储数据的仓库，数据库有组织存储数据。（DataBase DB）

数据库管理系统

​ 管理数据库的大型软件。（DataBase Management System DBMS)

SQL

​ 操作关系型数据库的编程语言，定义操作所有关系型数据库的统一标准。（Structured Query Language SQL结构化查询语言)
